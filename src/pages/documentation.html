Banana M-Tool

A tool for adding content to the website easily. Follow the instructions below to use it:

  • Open the M-Tool by clicking the button in the admin panel.
  • Choose the type of content you want to add (e.g., event, photograph, etc.).
  • Fill in the required fields with the content details.
  • Click "Submit" to add the content to the website.
